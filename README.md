Поддерживаемые игровые правила:
- [x] Чат-комната (聊天室)
- [x] Цзиньхуа (诈金花/三张牌)
- [x] Техасский Холдем (德州扑克)
- [ ] Дуйдичжу / Борьба с помещиком (斗地主)
- [ ] Блэкджек (21点)

Особенности: 
- [x] Кроссплатформенность: работает на Node.js, легко развертывается на различных платформах.
- [x] Масштабируемость: использование Redis в качестве шины сообщений и хранилища данных.
- [x] Открытая архитектура: благодаря Redis в качестве шины сообщений сервер легко интегрировать и расширять.
- [x] Кластеризация: использование PM2, липких сессий (sticky sessions) и socket.io-redis.
- [x] Протокол WebSocket: поставляется с клиентским API на JavaScript и веб-демо.
- [x] Регистратор событий для серверных событий и действий пользователей.

## Требования ##

* Redis

Redis — это высокопроизводительное хранилище данных структуры «ключ-значение» с открытым исходным кодом (лицензия BSD). Его часто называют сервером структур данных, поскольку ключи могут содержать строки, хэши, списки, множества, отсортированные множества, битовые карты и HyperLogLog.

[Скачать](http://redis.io/download)

* Node.js / npm

Node.js — это платформа, построенная на движке JavaScript Chrome V8, позволяющая легко создавать быстрые и масштабируемые сетевые приложения. Node.js использует событийно-управляемую модель неблокирующего ввода-вывода, что делает её легкой и эффективной, идеально подходящей для приложений реального времени с интенсивным обменом данными на распределенных устройствах.

[Скачать](http://nodejs.org/download)

* PM2 (для продакшна и кластеризации)

PM2 — это менеджер процессов для приложений Node.js со встроенным балансировщиком нагрузки. Он позволяет держать приложения активными постоянно, перезапускать их без простоев и облегчает выполнение стандартных задач системного администрирования.

[Подробнее](https://github.com/Unitech/pm2)

# Установка #

* Глобальная установка и запуск как службы с помощью PM2: 

```bash
[sudo] npm install pm2 -g
[sudo] npm install casino-server -g

# сначала запустите redis-server
redis-server &

# запуск службы в режиме кластера
pm2 start `which casino-server` -i 0
```

* Установка как локального приложения Node.js и запуск в текущей папке:

```bash
git clone https://github.com/developer3000S/casino-server-rus.git casino
cd casino
npm install
# sudo npm install -g gulp
# gulp build
node bin/casino-server [options]
```

Теперь вы можете открыть http://localhost:7000 — это веб-клиент игры для тестирования и демонстрации.

```bash
# откройте браузер для доступа к тестовой веб-странице
open http://localhost:7000/
```

# Доступные параметры запуска: #

-p Использовать указанный порт (по умолчанию 7000)

-h Использовать указанный хост (по умолчанию 0.0.0.0)

-r Адрес сервера Redis (по умолчанию 127.0.0.1:6379)

Запустите логгер событий для мониторинга:

```bash

```

Или записывайте события в файл логов:

```bash
bin/logger.js -o ./tmp/casino.log
```
